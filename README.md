# CITS3403_Group_Project
Group Project for CITS3403. Semester 1, 2026, UWA.

To set up virtual environment:
Make sure you are in the server directory, then
python3 -m venv .venv  (only do this if you don't already have .venv)
source .venv/bin/activate
in the terminal
and then install all dependencies
make sure all dependencies are updated in requirements.txt so venv can be initialised properly

To exit VENV:
deactivate
in the terminal

# CITS3403_Group_Project
Group Project for CITS3403. Semester 1, 2026, UWA.

To set up virtual environment:
Make sure you are in the flaskr directory, then in the terminal:
```
python3 -m venv .venv  (only do this first command if you don't already have .venv)
source .venv/bin/activate
```
and then install all dependencies from requirements.txt
make sure all dependencies are updated in requirements.txt so venv can be initialised properly
leave .venv in gitignore

To exit VENV:
```
deactivate
```
in the terminal

run the server in the terminal:
```
python app.py
```
